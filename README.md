# FillingOutAQuestionnaire

## Setup

- Nodejs >= v16 is installed `node -v`.
- Override question file in `src/assets/input.txt`
- run `npm install`

## Run

- run `npm start`
- type `http://localhost:4200/` in your browser
- Ctrl + C for shutdown

## Run tests

- run `npm test`
- it will open a browser window
- Ctrl + C to stop
